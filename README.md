# RMDH
Rate My Dining Hall is a full-stack web app that lets college students across the United States discuss and rate the dining halls and dining commons on their campus.

# About
RMDH is a personal project currently in progress. The goal is to create a platform where students can share opinions, reviews, and experiences about campus dining options in a simple and easy-to-use format.

# Features
- Rate dining halls and dining commons
- Leave discussions and reviews
- Browse campus dining locations
- Full-stack web application structure

# Tech Stack

- JavaScript
- HTML
- CSS

# Installation
`git clone https://github.com/LekhithP/RMDH.git`

`cd RMDH`

`npm install`

# Run The Project
`npm start`
